### 常用命令
- vim filename     // 创建文件
- Esc /i /a          // 在编辑模式之间切换
- i insert          // 在当前光标前面插入
- a append     // 在当前光标后面
- :w     // 写入保存
- :q  :wq :wq!     // 退出 并保存 强制
- yy/yw     // 复制  一行/一个词（空格区分一个词）
- p               // 粘贴
- dd/dw     // 删除     一行/一个词
- h/j/k/l     // 在非编辑模式下移动光标
- ^  &           // 移动到行首/行尾
- /关键字     // 查找内容
- :%s/关键词/替换词/gc          // %:所有，s:search，正则表达式，g:自动下一个，c:confirm确认
- :split/vsplit     // 分窗口
